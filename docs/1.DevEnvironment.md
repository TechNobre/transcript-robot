## Create a virtual environment:

```bash
python -m venv venv
```

## Activate the virtual environment:

```bash
.\venv\Scripts\activate
```

## Install the dependencies:

```bash
pip install -r requirements.txt
```

## Run the script:

```bash
python .\src\App.py
```
